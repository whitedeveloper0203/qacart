--- Install Guid ----
1. Unzip rar file witin a certain directory
2. Plz make sure you have installed Node.js in your computer.
3. Open the window command prompt and run below command
	> npm install -g yarn
4. In the command prompt, go to the directory what you have unziped and run below command
	1) Install npm packages according to the package.json 
		> yarn  * This is the yarn package install cmd.
	2) Run local server by running below command
		> yarn start

Then you can see running status in the command. 

After launching server, in the browser, you can see the site at http://localhost:3000
